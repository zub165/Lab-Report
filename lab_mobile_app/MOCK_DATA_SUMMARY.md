## 🔍 **How to View Your Mock Data**

### 1. **Interactive Data Viewer**
```bash
cd backend
python3 data_viewer.py
```

This provides an interactive menu to browse:
- Patients with medical history
- Tests with status and priority
- Test results (normal and abnormal)
- Appointments and payments
- Generated reports
- System statistics

### 2. **API Access**
Your mock data is accessible through the comprehensive API:

```bash
# Get authentication token
curl -X POST "http://localhost:8000/api/auth/login" \
  -H "Content-Type: application/json" \
  -d '{"username": "admin", "password": "admin123"}'

# View patients
curl -X GET "http://localhost:8000/api/data/patients" \
  -H "Authorization: Bearer YOUR_TOKEN"

# View tests with results
curl -X GET "http://localhost:8000/api/data/tests?include_results=true" \
  -H "Authorization: Bearer YOUR_TOKEN"

# View abnormal results
curl -X GET "http://localhost:8000/api/data/research/search?query=abnormal" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

### 3. **Interactive API Documentation**
Visit: `http://localhost:8000/docs`
- Test all endpoints directly
- View request/response examples
- Explore all available data
